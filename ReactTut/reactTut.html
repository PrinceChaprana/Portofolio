React Setup

  1. First create the folder for the project
  2. write code npm init -y in terminal
    and use ~ to open the terminal in vsCpde
  3. This will create package.json file in the project
    this contains the all frameworks like node expresss and ract that is used in the projects
  4. now Install react using
    npm install react react-dom in terminal
  5. to use JSX install babel
    npm install @babel/core @babel/preset-env @babel/preset-react babel-loader
    the @ is different packages from babel like core preset-env(for ES6) and react-preset(to convert JSX to react) babel-loader (to connect babel to webpack)
  6. create a babel file with name .babelrc
    now type the follwing
    {
    "presets" : ["@babel/preset-react","@babel/preset-env"]
    }
  7. Install webpack and connect above created babel to it
    npm install webpack webpack-cli webpack-dev-server
  8. to Connect the babel to webpack we need to create a file webpack.config.js
    and paste the following code
  9. install the webpack html to use the
    npm install html-webpack-plugin
  10. create folder src
    this will contain our react code
  11. create a App.js file in src folder this will code the components and file is
    import react from "react";
    const App = () => (
    Hello React
    );
    export default App;
  12. create a file index.js to render the above code
    import react from "react";
    import reactDom from "react-dom";
    import App form "./App";
    reactDom.render(< App/ >,document.querySelector("#root"));
  13. create a file index.html to base the above code
    with div id root
  14. To Run the code head to package.json file and replace line 7
    "start" : "webpack-dev-server --hot --open"
     (for developmment)
    "build" : "webpack --config webpack.config.js --mode producation"
    (for final build)
  15. npm install style-loader css-loader to use css in react
  16. to load images npm install file-loader
    and use the test to load the files
    ,
    {
    test:/\.png|jpg|svg|gif$/,
    use: ["file-loader"]
    }
